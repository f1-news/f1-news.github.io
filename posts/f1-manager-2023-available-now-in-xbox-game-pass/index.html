《F1 经理 2023》现已在 Xbox Game Pass 中推出

2023年10月19日 · 1 分钟 · 302 字

Frontier Developments 官方授权的一级方程式管理模拟游戏《F1 Manager 2023》将从今天起在 Xbox 和 PC 上的 Game Pass 订阅者中排队，让玩家能够完全控制一支官方 F1 车队，从工厂到前排。

在回归和改进的职业模式中，球队负责人将负责球队的每个领域。

在赛道上，动作以令人惊叹的广播式体验展开，通过包括新遮阳板摄像头在内的多个车载摄像机角度将每一个激动人心的时刻都栩栩如生，同时使用来自赛车工程师和车手的真实团队无线电消息来传达指令。

《F1 Manager 2023》的另一个新功能是全新的“比赛重播”模式，车迷将使用每场赛事的 F1 数据（例如发车位置和天气）改写 2023 年一级方程式世界锦标赛中每场真实比赛的结果

F1 Manager 2023 还可通过 Steam 和 Epic Games Store 在 PlayStation 4、PlayStation 5 和 PC 上下载。
