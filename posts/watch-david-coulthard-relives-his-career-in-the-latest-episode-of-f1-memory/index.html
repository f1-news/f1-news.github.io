Coulthard在最新一集的“ F1 Memory Box”中重现了他的职业生涯

2025年9月16日 · 1 分钟 · 307 字

F1 Memory Box推出了F1电视台，在F1电视台上提供了另一个新的独家系列，该盒子涉及这项运动的传说，通过以前看不见的镜头从他们的故事中分享了他们的故事。

苏格兰人从未见过的视频片段，他在回顾F1的旅程时从未见过的故事，该旅程跨越了近250场比赛，并在1990年代和2000年代获得了13个大奖赛。

一个杰出的例子（在下面的视频片段中）是库尔萨德（Coulthard）对坑停止混乱的回忆，而迈凯轮队在1998年澳大利亚大奖赛上局势局势，在那里，队友米卡·哈基宁（Mika Hakkinen）在有争议的情况下取得了胜利。
