诺里斯赞扬赞德沃特杆位的“惊人”表现，但预计家乡英雄维斯塔潘将在比赛当天“奋力拼搏”

    2024年8月24日 · 2 分钟 · 893 字

    兰多·诺里斯 (Lando Norris) 在从 F1 暑假归来后表现出色，表现出色，在荷兰大奖赛上取得了杆位，并取得了出色的单圈成绩，对此表示满意。

    下车回答媒体的第一个问题后，诺里斯说道：“真是令人惊奇的一天。很高兴能回来并以杆位起步。老实说，这是一圈不错的比赛。”

    “排位赛总是相当顺利，我跑出了一些不错的圈数，尤其是最后的那一圈，这始终是最重要的。车队做得很好，我对今天很满意。”

    在整个排位赛中，他都在充满挑战、大风的条件下航行，他继续说道：“每个人都在抱怨，因为这很棘手，这并不容易……每一圈你都必须知道会发生什么，知道你还能推动多少，极限在哪里是，但这归根结底就是我们的工作。

    “这并不容易，但我感觉很舒服，这辆车感觉很棒。我们一段时间以来第一次对赛车进行了一些升级，一切都运转良好，所以也非常感谢团队。”

    虽然诺里斯在迈阿密取得了他在 F1 比赛中的首场胜利，但这位英国人并没有将他之前的三个职业生涯杆位转化为任何杆位，包括本赛季早些时候的西班牙和匈牙利大奖赛。

    想起他在这两场比赛中的起步，考虑到他两次都失去了领先优势，诺里斯微笑着说：“我不知道你为什么这么说！我对明天感到很兴奋。

    “我确信这会很艰难。马克斯整个周末都很快。我知道我们今天得到了他，但他仍然是第二名，他将会打出一场精彩的战斗，特别是在他的主场比赛中。我正在寻找向前迈进。”

    诺里斯的队友奥斯卡·皮亚斯特里 (Oscar Piastri) 将从第三位发车开始周日的比赛，与他在第一季度和第二季度的努力相比，他在第三季度最后一圈的表现略显糟糕，他对此表示遗憾。

    “我真的做得不够好，”澳大利亚人承认。 “我认为前半圈相当稳定，然后后半圈就不太符合我的需要了。

    “这个周末这辆车跑得很快。没有更高一点有点令人失望，但我们下面仍然有一辆很好的赛车，我们可以获得一些非常好的积分，希望明天能获得一些奖杯。我们”来试试吧。”

    当被问及他是否认为自己仍然可以为胜利而战时，皮亚斯特里补充道：“我认为这绝对不是不可能的。从长远来看，节奏确实不错，过去几个周末我们的速度相当快。”

    “希望我们明天能度过美好的一天，有一个好的开始，如果可以的话，尽量弥补一些位置。明天会提供一些好东西。”
